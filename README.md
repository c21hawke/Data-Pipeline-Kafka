# STEP 1
```bash
conda create -p venv python=3.8 -y
```
# STEP 2
```bash
conda activate ./venv
```
# STEP 3
```bash
pip install -r requirements.txt
```
# STEP 4
```bash
touch .env
```
# STEP 5: Copy the below text and paste in .env file
```bash
API_KEY=""
API_SECRET_KEY=""
BOOTSTRAP_SERVER=""
SCHEMA_REGISTRY_API_KEY=""
SCHEMA_REGISTRY_API_SECRET=""
ENDPOINT_SCHEMA_URL=""
MONGO_DB_URL=""
```
